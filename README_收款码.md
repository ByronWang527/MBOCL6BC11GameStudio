# 添加您的个人收款码

## 需要的文件

为了使收款功能正常工作，您需要提供您的个人微信收款码图片：

- `wechat-pay-code.png` - 您的微信收款码图片

## 添加步骤

1. 准备您的微信收款码图片
2. 重命名为 `wechat-pay-code.png`
3. 将图片复制到项目的 `images` 目录中

图片文件应该放在这个路径：`/images/wechat-pay-code.png`

## 图片要求

- 建议尺寸: 300x300像素到500x500像素之间
- 格式: PNG或JPG文件
- 大小: 建议小于300KB，以确保快速加载

## 收款码将显示在以下位置

1. **游戏水印** - 游戏右下角的小水印
2. **吃豆人游戏试玩版** - 游戏预览界面右侧
3. **支付模态窗口** - 当用户点击购买时显示的大窗口

## 隐私和安全提示

- 此收款码会在网站上直接显示，可以被任何访问者看到
- 如果您将此网站发布到公共仓库，建议确保收款码图片不被包含在提交中
- 在自行部署时，请确保使用HTTPS协议以保护您的收款码和用户信息

## 重要提醒

收款码文件必须是 **真实的图片文件**，而不是文本文件。
如果您看到支付模态窗口中缺少收款码，请检查您是否已经正确添加了图片文件。
